
## How to run

Install packages

```bash
npm install 
```

run this command to generate csv file (output.csv)

```bash
  npm run dev
```

## Content of CSV File
- Column 1: User wallet address
- Column 2: etherum balance
- Column 3 to Column 5: token erc20 address
